2. Description of Service

Mistbot is a Discord bot built for the Attack on Titan: Revolution (Roblox) community. It provides the following services:

  • Vouch & Carrier Tracking — Log and display carries, leaderboards, and profiles.
  • Marketplace — Create, search, and manage in-game trade listings with a cross-server global feed.
  • Lobby System — Host and manage game lobbies with private channels and participant tracking.
  • AI Assistant — Answer game-related questions using AI-powered wiki search.
  • Mist-Connect — Automatically detect trade intent from messages in designated channels.
  • Security — Scam reporting, blacklisting, and appeal systems.
  • Values Lookup — Check in-game item values and compare trades.
